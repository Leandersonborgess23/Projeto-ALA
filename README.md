# Projeto-ALA
Repositório criado por Leanderson Borges para armazenar informações, dados e conteúdo do projeto ALA.

O projeto ALA (Automatização das Luzes do Auditório) insiste na implementação automática do jogo de luzes do auditório do IFRN campus Ceará-Mirim.

O projeto tem participação de Leanderson Borges, Diógenes Santiago e Miguel Carlos. E tem como orientador o docente Lennedy Campos Soares.
