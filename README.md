# Projeto-ALA
Repositório criado por Leanderson Borges para armazenar as informações, dados e conteúdo do projeto ALA.

O projeto ALA (Automatização das Luzes do Auditório) é um projeto de extensão que insiste na implementação automática do jogo de luzes do auditório do IFRN campus Ceará-Mirim, isso através da criação de um hardware, firmware e um software, onde podemos regular a intensidade da luz.

O projeto tem participação dos alunos de INFO3M: Leanderson Borges, Diógenes Santiago e Miguel Carlos. E tem como orientador o docente: Lennedy Campos Soares.
